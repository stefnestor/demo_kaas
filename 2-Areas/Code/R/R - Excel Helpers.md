## Net Present Value - NPV

````R
#' Net Present Value
#' 
#' @param rate Rate to use when discounting
#' @param values Vector of numeric values to discount to present value
npv <- function(rate, values) {
    sum(values / (1 + rate) ^ seq_along(values))
}
````
